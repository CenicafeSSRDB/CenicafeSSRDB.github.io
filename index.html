What is Cenicafé SSR Database?



It is an easy-to-use database to explore SSR microsatellite markers developed to characterise genetic diversity of different populations of organisms associated with coffee cultivation from the plant as pathogens and pests. In the Cenicafé SSRdb database, 6,674 SSRs of Coffea sp. that have been evaluated on acrylamide gels and amplify at one site in the genome or one site per subgenome and 5,469 SSR microsatellite markers with unique hit within the genome of H. vastatrix (Hv33) were identified.

This large number of SSR motifs identified were detected based on the genomes of Coffea arabica and H. vastatrix by successfully using the MISA (MIcroSA Satellite Identification Tool) pipeline (2015) and subsequently using the mod code (2019) and Primer3 version 4.1. 0 for the generation of primers from the flanking sequences, using codes for re-addressing the thermodynamic parameters which were as follows: melting temperature 50-65 °C, GC content between 40-70%, primer size 20-22 bp in length and product size 80-800 bp.

The SSRome database will be constantly updated. In the Cenicafé SSRdb database, main functional modules/pages were developed to cover all content of the identified SSR motifs, including "Database search", "Downloads"; and "Statistics".
